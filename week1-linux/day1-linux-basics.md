# Day 1 - Linux Basic Commands

## 1) pwd
**Meaning:** Shows current folder location (path).
**Syntax:** pwd
**Example 1:** pwd
**Example 2:** cd /home && pwd

---

## 2) ls
**Meaning:** Shows list of files/folders.
**Syntax:** ls
**Example 1:** ls
**Example 2:** ls -l

---

## 3) cd
**Meaning:** Used to change directory (folder).
**Syntax:** cd <folder_name>
**Example 1:** cd Downloads
**Example 2:** cd ..

---

## 4) mkdir
**Meaning:** Creates a new folder.
**Syntax:** mkdir <folder_name>
**Example 1:** mkdir devops
**Example 2:** mkdir week1 week2

---

## 5) touch
**Meaning:** Creates a new empty file.
**Syntax:** touch <file_name>
**Example 1:** touch notes.txt
**Example 2:** touch day1.md day2.md

---

## 6) rm
**Meaning:** Deletes a file/folder.
**Syntax:** rm <file_name>
**Example 1:** rm notes.txt
**Example 2:** rm -r week1-linux

---

## 7) clear
**Meaning:** Clears the terminal screen.
**Syntax:** clear
**Example 1:** clear
**Example 2:** clear
